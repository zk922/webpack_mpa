# 多页面通用webpack项目脚手架

通用的多页面webpack项目脚手架。项目结构：

```
|---- dist                    //打包输出目录
|---- scripts                 //webpack打包配置脚本
|---- scr                     //项目源码目录
       |---- components       //公共组件目录
       |---- env              //多环境配置目录
       |---- 
```
